Стартова веб аплікація

  1. Використовує встроений Apache Tomcat — контейнер сервлетів, розроблений Apache Software Foundation.
  2. Базується на Spring Framework Getting Started Securing a Web Application

Перший крок

Доведеня прикладу "Spring Framework Getting Started Securing a Web Application" до стану даної сторінки.

  1. Взяти приклад з: http://spring.io/guides/gs/securing-web/

    Потрібно тільки src директорія з complete директорії і build.gradle файл.

  2. Додати html сторінку src/main/webapp/page1/step1.html
  3. Зняти захист: "/page1/**" в WebSecurityConfig.java
  4. Зробити redirect на бажану сторінку.
    1. Зарееструвати template redirect.html в MvcConfig.java
    2. Налаштувати в redirect.html client-side redirect

      за прикладом http://www.w3.org/TR/WCAG20-TECHS/H76.html

  5. Компіляція: gradle clean build
  6. Запуск: java -jar build/libs/bug-cms-1-0.1.0.jar
  7. Альтернативний компіляція і запуск, за допомогою Spring Tool Suite™
  8. https://github.com/romanm/bugCms1_step1
